# Contributing

Thanks for your suggestions!

## Guidelines

Please ensure your pull request adheres to the following:

- Search previous entries before making a new one, as yours may be a duplicate.
- Make an individual pull request for each suggestion.
- Use the table format: `| [**Tool Name**](link) | Description. |`
- Keep descriptions short and simple, but descriptive.
- Do not use articles (a, an, the) at the start of descriptions.
- End all descriptions with a full stop/period.
- Do not use emojis.
- New categories or improvements to existing categorization are welcome.
- Add only tools and resources you truly like.
- New links should be added below existing entries in each category.
- Check your spelling and grammar.
- Make sure your text editor is set to remove trailing whitespace.

## What We're Looking For

- Developer tools and infrastructure
- Design resources and component libraries
- AI/ML tools and platforms
- Productivity and business tools
- Agencies and designers with proven track records

## What We're Not Looking For

- Affiliate or referral spam
- Tools that require sign-up just to see what they do
- Abandoned projects with no recent updates
- Duplicates of existing entries

## Questions?

Open an issue if you're unsure whether something fits.
